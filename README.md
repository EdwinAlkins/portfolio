# Portfolio

Personal portfolio website built with **React** and **TypeScript**.

🔗 [Live Demo](https://edwinalkins.github.io/portfolio/)

## Tech Stack

- **React 19** + **TypeScript**
- **styled-components** for styling
- **react-router-dom** for routing
- **cmdk** for command palette navigation
- **Vite** for bundling

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## PDF Generation

Generate a PDF version of the CV using Puppeteer:

```bash
npm run generate-pdf
```

This script converts the HTML CV to PDF and outputs it to `public/pdf/cv-william-nauroy.pdf`.

## Deployment

Deployed to GitHub Pages:

```bash
npm run deploy
```
